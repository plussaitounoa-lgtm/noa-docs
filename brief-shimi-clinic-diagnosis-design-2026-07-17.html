③ 設問パネル（5枚）
#質問選択肢
Q1性別を教えてください女性 / 男性
Q2あなたの年代は？20代 / 30代 / 40代 / 50代以上
Q3現在抱えているお悩みは？（非分岐・エコー用）はっきりした茶色いシミ / 頬にモヤっと広がるシミ・くすみ / 小さいシミ・そばかすが多数ある / 複数当てはまる
Q4シミ取りで期待することは？（分岐軸1）気になる一点をしっかり消したい / 顔全体のくすみもケアしたい / とにかくまとめて全部消したい
Q5クリニックを選ぶなら、一番重視するのは？（分岐軸2・最終設問）できるだけ費用を抑えたい / 通いやすさ・予約のしやすさ / 万が一のときの対応・アフターケア / じっくり相談してから決めたい
Q3は振り分けに使わない非分岐設問。専門用語（ピコトーニング等）は設問側では一切出さず、この後のPRカルーセルで初めて開示する。

⑤ ピコトーニングって？5枚
ピコトーニング｜ピンク系（#d6336c）
Card役割内容
1説明ピコトーニングとは。メラニン色素へ集中的にレーザーを照射し、色素を粉砕・除去。ピコ秒照射のため組織へのダメージが少なく、炎症後の色素沈着も起こりにくい
2図解従来レーザー vs PICO LASERの粉砕度合いの比較図（表皮/真皮の断面イラスト）
3おすすめ顔まわりの肌悩みイメージ写真＋チェックリスト4項目（肝斑／シミ・くすみ／毛穴の開き／トーンアップ）
4症例紹介Before/After写真＋限定解除要件4点併記（治療内容／治療期間・回数／費用／主なリスク・副作用）。写真は実物差し替えが必要
5キャンペーン+CTAライト1回¥9,800→初来院クーポンで¥0。無料カウンセリング予約CTA
⑥ ピコスポットって？5枚
ピコスポット｜ティール・ブルー系（#04b6b4 / #0f95da）
Card役割内容
1説明ピコスポットとは。少ない照射回数で気になる一点にピンポイントでアプローチ。薄いシミなら1回で除去できる場合も
2図解ナノ秒パルス vs ピコ秒パルスの熱影響層の比較図
3おすすめチェックリスト4項目（濃いシミ／そばかす／ピンポイント／1回でしっかり改善）
4🔧 修正中症例紹介から施術後のダウンタイムを説明するカードに差し替え予定（症例素材が使用できないため）。内容確定後に共有するため、製作は後回しでOK
5キャンペーン+CTAシミ取り1mm以下¥6,600→初来院クーポンで¥0。無料カウンセリング予約CTA
ピコトーニングと構成・トンマナは同一、配色のみブランドカラーを変更。Card4は修正中のため、Card1〜3・5から先に着手可。

公式LPのQ&Aをそのまま転載せず、ユーザー目線の見出し回答型ガイドとして再構成（Qタグ→見出し回答→写真→解説文の順）。メインカラー（水色）を使用。

Q3ダウンタイムはどのくらい？ほとんど気になりません

⑧ 施術までの流れを見てみる5枚
STEP見出し内容
01カウンセリング予約カウンセリングは無料。Web・LINE・電話で予約OK。施術するか迷っている段階でも気軽に相談できます
02美容カウンセラーとのカウンセリング来院当日、専任カウンセラーが悩みや不安をヒアリング。ドクターに聞きにくいことや料金の支払い方法など不明点を解消できます
03担当ドクターとのカウンセリングシミュレーションから施術まで同じドクターが担当。最適なプランを提案してくれます。シミュレーション画像は持ち帰りOK！
04ご契約プランやダウンタイム、リスクに納得できたら契約。空きがあれば当日施術も可能です
05術後のアフターケア施術後のケア方法を説明。必要に応じて後日の経過観察もあります
全5枚に画像・イラストのプレースホルダーあり（差し替え予定）。TCB公式LP「施術の流れ」の内容をユーザー目線でコンパクトに要約したもの。

診断設計（Q1〜Q5）

所要時間: 約30秒（5問）／ Q5の回答で結果タイプが分岐

Q1 — データ収集
あなたの裸眼視力はどのくらいですか？
視力レベル把握・ユーザー分析 / 矯正方法分岐の精度向上
  • A 裸眼だと数メートル先がほぼ見えない（0.05未満）→ ICL濃厚
  • B 手元は見えるが1〜2m先の文字が読めない（0.05〜0.1）→ ICL/レーシック両候補
  • C 遠くの看板・黒板が読めない（0.1〜0.3）→ レーシック寄り
  • D 少し悪い程度、日常はなんとかなる（0.3以上）→ 眼鏡・コンタクト寄り
  • E わからない / 測ったことがない→ 中間扱い
Q2
今、視力矯正をどうしていますか？
最軽入口・自分ごと化
  • A コンタクトレンズを使っている
  • B メガネを使っている
  • C コンタクトとメガネを使い分けている
  • D 何もしていない（裸眼）
Q3
目のことで一番困っていることは？
ペイン把握・自分ごと化
  • A 視力がどんどん悪くなっている気がする
  • B コンタクト・メガネが面倒・不便
  • C 目が疲れやすい・乾く
  • D 左右で見え方が違う・ピントが合いにくい
Q4
もし目の悩みが解決したら、一番変わってほしいことは？
理想の未来・感情接続
  • A 朝起きたらすぐ見える生活がしたい
  • B スポーツや旅行・温泉を、コンタクトやメガネなしで楽しみたい
  • C ドライアイや目の疲れから解放されたい
  • D 目のことを気にせず過ごしたい
Q5 — 分岐キー
視力矯正に求めることで、一番近いのは？
矯正方法の直指定 → 結果タイプ分岐
  • A 今のスタイルのまま、もっと快適に管理したい → 眼鏡タイプ
  • B メガネなしで動き回れる自由が欲しい → コンタクトタイプ
  • C 根本から解決したい。コスパ重視 → レーシックタイプ
  • D 根本から解決したい。目への負担を最小限にしたい → ICLタイプ

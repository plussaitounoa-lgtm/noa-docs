3. 設問パネル（5枚）

Q1 · 視力レベル把握
あなたの裸眼視力はどのくらいですか？
  • A 0.05未満
  • B 0.05〜0.1
  • C 0.1〜0.3
  • D 0.3以上
Q2 · 現在の矯正状況
今、視力矯正をどうしていますか？
  • A コンタクトレンズを使っている
  • B メガネを使っている
  • C コンタクトとメガネの併用
  • D 何もしていない（裸眼）
Q3 · ペイン把握
目のことで一番困っていることは？
  • A 視力がだんだん落ちている
  • B コンタクト・メガネが面倒・不便
  • C 目が疲れやすい・乾く
  • D 左右差やピントが合わない
Q4 · 感情接続
もし目の悩みが解決したら、一番変わってほしいことは？
  • A 朝起きたらすぐ見える生活
  • B メガネなしで運動や旅行をする
  • C ドライアイや目の疲れから解放
  • D 目のことを気にせず過ごしたい
Q5 · 分岐キー ⭐
視力矯正に求めることで、一番近いのは？
この回答のみで結果タイプが決定
  • A 今より見えやすくなること → TYPE A（眼鏡）
  • B 眼鏡の不便さと見た目をなくす → TYPE B（コンタクト）
  • C 眼鏡やコンタクトから解放されること → TYPE C（ICL/レーシック）

① PUバナー（3枚）
ヘッドラインあなたの肌タイプ診断
案Aその化粧品、自分の肌タイプに合わせて選べてる？
案Bいろいろ試してるのに変わらないなら、肌タイプから見直してみない？
案C自分の肌タイプ、ちゃんと知りたい！

/カンタン3問診断　ボタン: 「はい」 / 「いいえ」

使い分けの目安: ファンデ/BB比較記事（商品比較で疲れている層）は案B。美容オイル記事（単品への興味段階）は案A。成分解説記事（知識収集・能動的な層）は案C。
② 診断表紙（1枚）

タイトル: あなたの肌タイプ診断

サブ: 肌タイプに合わせた3ステップのおすすめケアがわかる

本文: 化粧品選びで迷子になるのは、自分の肌タイプに合わせて選べていないから。3問でわかる無料診断。

バッジ: 匿名・無料

③ 設問パネル（3枚）
#質問選択肢
Q1
振り分け主軸		洗顔後15分、何もつけずにいると肌は？A: 突っ張る、粉をふく →乾燥肌 ／ B: 顔全体がテカって皮脂が浮く →脂性肌 ／ C: おでこ・鼻はテカるのに頬は突っ張る →混合肌 ／ D: 赤みが出たり、ピリピリ・かゆみが出る →敏感肌
Q2新しいコスメを使い始めたとき、肌に出やすい反応は？A: 特に何も起きない ／ B: 赤み・かゆみが出ることがある ／ C: 何を使っても変化を感じにくい ／ D: ニキビ・吹き出物が増えることがある
Q3一番気になる肌悩みは？A: 乾燥・カサつき ／ B: 毛穴・テカリ ／ C: 崩れやすさ・肌の揺らぎ ／ D: 赤み・刺激に弱い
Q1が診断の振り分け主軸（1問で4タイプ確定）。Q2・Q3は非分岐で、Q1の判定を裏付ける確認軸兼、結果カルCard2のパーソナライズ用。
④ 結果カルーセル（16枚）
乾燥肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「乾燥肌」 / ☑ 洗顔後すぐ、肌がつっぱる / ☑ 化粧水をつけても、乾いた感じがすぐ戻る / ☑ 季節の変わり目に肌が荒れやすい / ☑ 粉をふくことがある
2根本示唆＋矢印図式うるおいを与えても乾燥が戻る、本当の理由。 / 【矢印図】肌のバリア機能が乱れている → 角層の水分保持力が低下する → 与えたうるおいもすぐ蒸発する → 乾燥が進む / うるおいを「届ける」だけでなく「留める」ケアが必要！
3アプローチ3ステップうるおいを届けて、逃さないケアがカギ / ✓ クレンジング → 洗いながらもうるおいを守るタイプへ / ✓ スキンケア → バリア機能を立て直すタイプへ / ✓ 化粧水 → 角層の奥までうるおいを届けて留めるタイプへ / 3ステップ全部を「保湿」の視点で揃えるアプローチ。
4未来＋メリット列挙＋CTAうるおいのバリアが整うだけで、肌が変わる。 / ✓ つけた瞬間、じゅわっとうるおいが入る / ✓ 季節の変わり目でも肌が揺らぎにくくなる / ✓ もちっとした肌感が続く / うるおいに満ちた、乾燥知らずの肌へ。 / ボタン「あなたに合うケアを見る →」
脂性肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「脂性肌」 / ☑ 顔全体がすぐテカる / ☑ 皮脂が浮いてくる感じがある / ☑ 毛穴が目立ちやすい / ☑ 洗浄力を上げてもすぐ皮脂が戻る
2根本示唆＋矢印図式洗ってもすぐテカる、本当の理由。 / 【矢印図】洗浄力を上げて皮脂を取りすぎる → 肌が乾燥から守ろうとする → 皮脂の分泌がさらに増える → もっとテカる / 「しっかり落とす」と「乾燥させない」を両立させることが必要！
3アプローチ3ステップ皮脂と上手に付き合うケアがカギ / ✓ クレンジング → 毛穴の汚れまでしっかり落とすタイプへ / ✓ スキンケア → 皮脂の分泌を根本からコントロールするタイプへ / ✓ 化粧水 → 毛穴を引き締めながら整えるタイプへ / 「絶妙なバランス」で3ステップを揃えるアプローチ。
4未来＋メリット列挙＋CTA皮脂のリズムが整うだけで、肌が変わる。 / ✓ 鏡を見ても毛穴が気にならなくなる / ✓ 皮脂によるメイク崩れが減る / ✓ キメが整って、つるんとした質感になる / 皮脂に振り回されない、引き締まった肌へ。 / ボタン「あなたに合うケアを見る →」
混合肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「混合肌」 / ☑ おでこ・鼻はテカるのに、頬は乾燥する / ☑ 部分によって肌の調子が違う / ☑ オールインワンだと物足りなさを感じる / ☑ 季節で崩れる場所が変わる
2根本示唆＋矢印図式部分ごとに違う肌、本当の理由。 / 【矢印図】皮脂腺の分布が部位ごとに違う → Tゾーンは皮脂が出やすく頬は乾燥しやすい → 1つの製品で全部をまかなおうとする → どこかに無理が出る / 肌全体を整える視点でケアすることが必要！
3アプローチ3ステップバランスを整えるケアがカギ / ✓ クレンジング → 摩擦レスで部分ごとの負担を減らすタイプへ / ✓ スキンケア → 肌全体のバリア機能を整えるタイプへ / ✓ 化粧水 → 肌全体にムラなくうるおいを届けるタイプへ / 「バランス」で3ステップを揃えるアプローチ。
4未来＋メリット列挙＋CTA肌全体のバランスが整うだけで、肌が変わる。 / ✓ 部分ごとの調子の差が気にならなくなる / ✓ 崩れやすさが落ち着く / ✓ 肌全体がワントーンに見える / どこも無理していない、整った肌へ。 / ボタン「あなたに合うケアを見る →」
敏感肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「敏感肌」 / ☑ 新しいコスメで赤み・かゆみが出ることがある / ☑ 季節の変わり目に肌が揺らぎやすい / ☑ 洗顔後、ヒリつくことがある / ☑ 刺激の強いものは避けている
2根本示唆＋矢印図式また揺らいだ、本当の理由。 / 【矢印図】バリア機能が弱っている → 刺激をブロックしきれない → 成分にすら過敏に反応する → 赤み・かゆみが出る / 刺激をゼロに近づけながら、バリアを育てることが必要！
3アプローチ3ステップ刺激の少なさを軸にしたケアがカギ / ✓ クレンジング → 肌を傷めずやさしく落とすタイプへ / ✓ スキンケア → 刺激ゼロで肌環境ごと整えるタイプへ / ✓ 化粧水 → 低刺激設計で揺らぎを落ち着かせるタイプへ / 「刺激の少なさ」で3ステップを揃えるアプローチ。
4未来＋メリット列挙＋CTAバリアが育つだけで、肌が変わる。 / ✓ 新しいものを試すのが怖くなくなる / ✓ 季節の変わり目でも荒れにくくなる / ✓ 肌が安定している日が増える / 安心してつけられる毎日へ。 / ボタン「あなたに合うケアを見る →」
結果カード（4枚×4タイプ）には商材名・価格は一切出しません。Card4のCTA「あなたに合うケアを見る→」タップ後、おすすめ3点セットカルで初めて具体的な商品が登場します。

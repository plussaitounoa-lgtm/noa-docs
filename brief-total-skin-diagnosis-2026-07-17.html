対象: ①クレンジング診断②スキンケア診断③化粧水診断のどれにも属さない記事（ファンデ/BB比較・美容オイル・成分解説等）への横断入口診断。診断結果は「別の診断への誘導」ではなく、その場でクレンジング・スキンケア・化粧水3ステップ分の商品を直接レコメンドする一段階構造。①②③の既存商材（アフィリリンク含む）をそのまま流用するため新規のアフィリ提携は不要。
2026-07-21確定版： 当初の全3問構成は「3問で何がわかるのか、設問が少なすぎて説得力に欠ける」とNoaに指摘され、競合4社（Dr.サポーター/AYAKA/津田コスメ/DHC）リサーチ＋対象読者ユーザーダイブを実施。一度「複数設問のAND判定・多数決」で設計したがシステム制約NG-2（診断結果は最終設問の回答のみで分岐する）に抵触し実装不可と判明。「Q5で仮判定→Q6でグループ別に確定」という2段階アダプティブ分岐（NG-2準拠）に組み直し、非分岐4問→分岐2問の全6問に確定。

① PUバナー（3枚）
ヘッドラインあなたの肌タイプ診断
案Aその化粧品、自分の肌タイプに合わせて選べてる？
案Bいろいろ試してるのに変わらないなら、肌タイプから見直してみない？
案C自分の肌タイプ、ちゃんと知りたい！

/カンタン6問診断　ボタン: 「はい」 / 「いいえ」

配信方法: 3案は記事タイプ別の決め打ち振り分けではなく、①②③と同じく全対象記事に均等にABテストで出す運用。以下は各案を用意した際の読者心理の参考情報（ファンデ/BB比較記事＝商品比較で疲れている層を想定した案B、美容オイル記事＝単品への興味段階を想定した案A、成分解説記事＝知識収集・能動的な層を想定した案C）。
② 診断表紙（1枚）

タイトル: あなたの肌タイプ診断

サブ: 肌タイプに合わせた3ステップのおすすめケアがわかる

本文: 化粧品選びで迷子になるのは、自分の肌タイプに合わせて選べていないから。6問でわかる無料診断。

バッジ: 匿名・無料

③ 設問パネル（6枚：非分岐4問＋第一分岐1問＋第二分岐1問）
#質問選択肢
Q1
非分岐		年代を教えてくださいA: 10〜20代 ／ B: 30代 ／ C: 40代以上
Q2
非分岐・根拠づけ		お化粧をしていて、崩れ方で当てはまるのは？A: 乾燥でカサついて粉をふいてくる ／ B: 皮脂で全体的にドロドロ崩れる ／ C: Tゾーンだけ崩れて頬はそのまま ／ D: 新しいコスメだと崩れ方がいつもと違う
Q3
非分岐・根拠づけ		毛穴の目立ちやすい場所は？A: 目立たない ／ B: 頬・フェイスライン全体 ／ C: Tゾーン（おでこ・鼻） ／ D: 頬・口周りなど乾燥ゾーン
Q4
非分岐・根拠づけ		一番気になる肌悩みは？A: 乾燥・カサつき ／ B: 毛穴・テカリ ／ C: 崩れやすさ・肌の揺らぎ ／ D: 赤み・刺激に弱い
Q5
第一分岐・グループ仮判定		洗顔後15分、何もつけずにいると肌は？A: 突っ張る、粉をふくことがある →乾燥・敏感グループ ／ B: 顔全体がテカって皮脂が浮く →脂性・混合グループ ／ C: おでこ・鼻はテカるのに頬は突っ張る →脂性・混合グループ ／ D: 赤みが出たり、ピリピリ・かゆみが出る →乾燥・敏感グループ
Q6
第二分岐・最終設問＝確定		Q6-①（Q5でA/D）: 新しいコスメを使うと、赤みやかゆみが出やすいですか？　→はい:敏感肌／いいえ:乾燥肌
Q6-②（Q5でB/C）: 頬やフェイスラインは、テカりより乾燥が気になりますか？　→いいえ:脂性肌／はい:混合肌		（2パターン、Q5の回答で自動的にどちらかが表示）
当初「Q2〜Q4の3問をAND判定・多数決」で設計したが、システム制約NG-2（診断結果は最終設問の回答のみで分岐する。複数設問の統合は不可）に抵触し実装不可と判明。Q5で肌質を仮判定→Q6でグループ別に確認・確定という2段階のアダプティブ分岐に組み直した（Q6の回答のみで確定するためNG-2準拠）。Q1〜Q4は非分岐（結果に紐づかない）で、Card2のパーソナライズ・根拠づけ用。特にQ2は他社Dr.サポーターの「メイクの崩れ方」軸で対象記事（ファンデ/BB比較記事）の読者に刺さりやすい。
④ 結果カルーセル（16枚）
2026-07-23改訂： Card3は当初「アプローチ3ステップ」（クレンジング/スキンケア/化粧水を1枚に列挙）だったが、総合診断としての差別化は3点セットカル（3カテゴリの商品が並ぶ場所）で果たされるため、結果カード側で無理に3カテゴリの型を持たせる必要はないと判断。①②③のCard3パターン（成分リスト）に立ち返り「おすすめケア＋向いてる成分＋NG」形式に統一。脂性肌・混合肌・敏感肌のCard1チェックリストも対象記事の文脈に合わせて差し替え済み。
乾燥肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「乾燥肌」 / ☑ 洗顔後すぐ、肌がつっぱる / ☑ 化粧水をつけても、乾いた感じがすぐ戻る / ☑ 季節の変わり目に肌が荒れやすい / ☑ 粉をふくことがある
2根本示唆＋矢印図式うるおいを与えても乾燥が戻る、本当の理由。 / 【矢印図】肌のバリア機能が乱れている → 角層の水分保持力が低下する → 与えたうるおいもすぐ蒸発する → 乾燥が進む / うるおいを「届ける」だけでなく「留める」ケアが必要！
3おすすめケアうるおいを届けて、逃さないケアがカギ / おすすめケア：✓ 化粧水は1〜2度重ね付けする ✓ 仕上げは乳液・クリームで蓋をする ✓ 洗浄力の強すぎない洗顔料を選ぶ / 向いてる成分：セラミド・ヒアルロン酸・シアバター / NG：洗浄力の強すぎる洗顔・スクラブでのゴシゴシ摩擦
4未来＋メリット列挙＋CTAうるおいのバリアが整うだけで、肌が変わる。 / ✓ つけた瞬間、じゅわっとうるおいが入る / ✓ 季節の変わり目でも肌が揺らぎにくくなる / ✓ もちっとした肌感が続く / うるおいに満ちた、乾燥知らずの肌へ。 / ボタン「あなたに合うケアを見る →」
脂性肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「脂性肌」 / ☑ 顔全体がすぐテカる / ☑ 皮脂が浮いてくる感じがある / ☑ 日中、ファンデがドロドロに崩れる / ☑ 洗浄力を上げてもすぐ皮脂が戻る
2根本示唆＋矢印図式洗ってもすぐテカる、本当の理由。 / 【矢印図】洗浄力を上げて皮脂を取りすぎる → 肌が乾燥から守ろうとする → 皮脂の分泌がさらに増える → もっとテカる / 「しっかり落とす」と「乾燥させない」を両立させることが必要！
3おすすめケア皮脂と上手に付き合うケアがカギ / おすすめケア：✓ 皮脂を落としすぎない、程よい洗浄力を選ぶ ✓ さっぱりタイプでもしっかり水分補給する ✓ 皮脂を吸着するテクスチャーを活用する / 向いてる成分：ビタミンC・ナイアシンアミド・サリチル酸(BHA) / NG：アルコール強めケア・過剰な洗顔
4未来＋メリット列挙＋CTA皮脂のリズムが整うだけで、肌が変わる。 / ✓ 鏡を見ても毛穴が気にならなくなる / ✓ 皮脂によるメイク崩れが減る / ✓ キメが整って、つるんとした質感になる / 皮脂に振り回されない、引き締まった肌へ。 / ボタン「あなたに合うケアを見る →」
混合肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「混合肌」 / ☑ おでこ・鼻はテカるのに、頬は乾燥する / ☑ 部分によって肌の調子が違う / ☑ ファンデが均一に馴染まない / ☑ 季節で崩れる場所が変わる
2根本示唆＋矢印図式部分ごとに違う肌、本当の理由。 / 【矢印図】皮脂腺の分布が部位ごとに違う → Tゾーンは皮脂が出やすく頬は乾燥しやすい → 1つの製品で全部をまかなおうとする → どこかに無理が出る / 肌全体を整える視点でケアすることが必要！
3おすすめケアバランスを整えるケアがカギ / おすすめケア：✓ Tゾーンと頬で使用量を使い分ける ✓ 季節に合わせて保湿量を調整する ✓ 全顔一律のオールインワンに頼りすぎない / 向いてる成分：セラミド・CICA・ナイアシンアミド / NG：顔全体を同じケアで固定する
4未来＋メリット列挙＋CTA肌全体のバランスが整うだけで、肌が変わる。 / ✓ 部分ごとの調子の差が気にならなくなる / ✓ 崩れやすさが落ち着く / ✓ 肌全体がワントーンに見える / どこも無理していない、整った肌へ。 / ボタン「あなたに合うケアを見る →」
敏感肌（4枚）
Card役割テキスト
1タイプ発表＋チェックリストあなたは「敏感肌」 / ☑ 新しいコスメで赤み・かゆみが出ることがある / ☑ 季節の変わり目に肌が揺らぎやすい / ☑ 洗顔後、ヒリつくことがある / ☑ 話題の成分を試すと、逆に肌荒れすることがある
2根本示唆＋矢印図式また揺らいだ、本当の理由。 / 【矢印図】バリア機能が弱っている → 刺激をブロックしきれない → 成分にすら過敏に反応する → 赤み・かゆみが出る / 刺激をゼロに近づけながら、バリアを育てることが必要！
3おすすめケア刺激の少なさを軸にしたケアがカギ / おすすめケア：✓ シンプルケアを優先する ✓ 低刺激・無添加のアイテムを選ぶ ✓ 摩擦を避けるやさしい使い方を意識する / 向いてる成分：CICA・セラミド・アラントイン / NG：スクラブ・強いピーリング・新商品のいきなり大量使用
4未来＋メリット列挙＋CTAバリアが育つだけで、肌が変わる。 / ✓ 新しいものを試すのが怖くなくなる / ✓ 季節の変わり目でも荒れにくくなる / ✓ 肌が安定している日が増える / 安心してつけられる毎日へ。 / ボタン「あなたに合うケアを見る →」

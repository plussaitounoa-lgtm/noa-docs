① PUバナー（3枚）
ヘッドラインあなたの化粧水タイプ診断
案A何を選べばいいか分からないのは、悩みの方向性が定まってないだけ。
案Bその化粧水、悩みの方向性は合ってる？5問でタイプがわかる。
案C私にぴったりの化粧水タイプが知りたい！

/カンタン5問診断
ボタン: 「はい」 / 「いいえ」

使い分けの目安: 「化粧水 効果ない／合わない」系記事は案A。「化粧水 選び方／おすすめ」系記事は案B。「毛穴 化粧水」「毛穴 引き締め」系記事にも同じ文言がそのまま使えます（タイプD＝毛穴ケア型が受け皿のため、記事別の追加コピーは不要）。
② 診断表紙（1枚）
タイトルあなたの化粧水タイプ診断
サブあなたに合う化粧水の悩みの方向性がわかる
本文化粧水選びで迷子になるのは、悩みの方向性が定まっていないから。5問でわかる無料診断。
バッジ匿名・無料
③ 設問パネル（5枚）
#質問選択肢
Q1あなたの年代を教えてください20代 / 30代 / 40代 / 50代以上
Q2普段の肌の状態は？カサつく＋乾燥しがち / ベタつく / ベタつく＋カサつきも / あまり気にならない
Q3洗顔後しばらくすると、肌はどんな感じ？つっぱりを感じる / 肌がゆるんでハリがない感じがする / 顔全体がくすんで見える気がする / 毛穴が目立つ・皮脂がにじんでくる感じがする
Q4季節の変わり目に、肌の状態は変わりますか？肌荒れしがち / ゆらぎやすい / 変わらない
Q5一番気になる肌悩みは？（最終設問・タイプ確定）乾燥・つっぱり感、うるおい不足 →タイプA / ハリ低下・たるみ・年齢による変化 →タイプB / シミ・くすみ・透明感のなさ →タイプC / 毛穴の目立ち・皮脂の多さ →タイプD
Q1〜Q4は非分岐（結果に紐づかない）。Q5（一番気になる悩み）が診断の最終設問であり、かつタイプを確定する分岐点そのものです。
④ 結果カルーセル16枚
タイプA｜うるおいチャージ型
Card役割テキスト
1タイプ発表＋チェックリストあなたは「うるおいチャージ型」 / ☑ 洗顔後すぐにつっぱりを感じる / ☑ 化粧水をつけてもすぐ乾いた感じに戻る / ☑ 季節の変わり目に、人より肌が荒れやすい / ☑ 夜になると肌がかさついてゴワつく
2根本示唆＋矢印図式保湿してるのに、肌が乾いていく。 / うるおいが逃げていく、本当の理由。 / 【矢印図】肌のバリア機能が乱れている → 角層の水分保持力が低下する → 与えたうるおいもすぐ蒸発する → 乾燥が進む / うるおいを内側までチャージすることが必要！
3解決策＋成分リストうるおいの土台づくりがカギ / ✓ ヒアルロン酸 → 水分を抱え込んで肌に届ける / ✓ セラミド → うるおいを外に逃がさないバリアを作る / ✓ グリセリン → 角層にうるおいを引き込んで乾燥を防ぐ / うるおいを与えて、閉じ込めるアプローチ。
4未来＋メリット列挙＋CTAうるおいのバリアが整うだけで、肌が変わる。 / 肌のバリア機能が整うと… / ✓ じゅわっとうるおいが入って、つっぱらなくなる / ✓ 季節の変わり目でも肌が揺らぎにくくなる / ✓ もちっとした肌感が続く / うるおいに満ちた、乾燥知らずの肌へ。 / ボタン「あなたに合う化粧水を見る →」
タイプB｜ハリ・エイジングケア型
Card役割テキスト
1タイプ発表＋チェックリストあなたは「ハリ・エイジングケア型」 / ☑ 肌がゆるんでハリがない感じがする / ☑ 頬や輪郭のもたつきが気になる / ☑ 季節の変わり目に、肌の調子が急に落ちる / ☑ 若い頃と同じケアなのに、ハリの戻りが遅くなった
2根本示唆＋矢印図式そのハリ低下、年齢だけじゃないかも。 / ハリが戻らない、本当の理由。 / 【矢印図】年齢とともに真皮でのコラーゲン・エラスチンの生成力が低下する → 肌を支える土台が弱くなる → ハリが戻らず、ゆるみが定着する / 肌の内側から弾力を立て直すことが必要！
3解決策＋成分リストハリを立て直す成分選びがカギ / ✓ レチノール → ハリを生み出す力を立て直す / ✓ ペプチド → コラーゲンの生成を促す / ✓ ビタミンC誘導体 → 肌のキメとハリの土台を整える / 肌の内側から立て直すアプローチ。
4未来＋メリット列挙＋CTA肌の土台が整うだけで、肌が変わる。 / 肌の土台が立て直されると… / ✓ 指で押すとふっと押し返す、ハリの違いを感じる / ✓ 老けた印象を言われなくなる / ✓ 季節の変わり目でも肌が乱れにくくなる / 老けを感じさせない、ハリのある肌へ。 / ボタン「あなたに合う化粧水を見る →」
タイプC｜美白・透明感型
Card役割テキスト
1タイプ発表＋チェックリストあなたは「美白・透明感型」 / ☑ 顔全体がなんとなくくすんで見える / ☑ シミが少しずつ増えてきた気がする / ☑ スキンケアを続けても透明感が出ない / ☑ 何を試しても効果を感じられなかった
2根本示唆＋矢印図式そのくすみ、「効果なし」で終わらせないで。 / くすみが取れない、本当の理由。 / 【矢印図】紫外線・刺激でメラニンが過剰に作られる → 排出されずに肌に蓄積する → くすみ・シミとして残る / メラニンを抑えて、外に出すことが必要！
3解決策＋成分リストメラニンを抑えて排出することがカギ / ✓ ビタミンC誘導体 → メラニンの生成を抑え、透明感を引き出す / ✓ トラネキサム酸 → メラノサイトの過剰な反応を抑える / ✓ アルブチン → メラニンの生成そのものを抑制する / メラニンにアプローチして、明るさを引き出す。
4未来＋メリット列挙＋CTAメラニンの生成を抑えるだけで、肌が変わる。 / メラニンの生成が抑えられると… / ✓ くすみが薄れて、光の反射が変わる / ✓ 「肌きれいになった?」と言われる日が来る / ✓ 試行錯誤を繰り返さなくなる / 澄んだ透明感がある肌へ。 / ボタン「あなたに合う化粧水を見る →」
タイプD｜毛穴ケア型 — 新規タイプ
Card役割テキスト
1タイプ発表＋チェックリストあなたは「毛穴ケア型」 / ☑ 毛穴の開き・目立ちが気になる / ☑ 皮脂が多くテカりやすい / ☑ 洗顔後すぐにまた皮脂が出てくる / ☑ 洗浄力を上げても毛穴が改善しない
2根本示唆＋矢印図式その毛穴、洗浄力のせいじゃないかも。 / 毛穴が目立つ、本当の理由。 / 【矢印図】肌が乾燥やストレスを感知すると皮脂腺が過剰に反応する → 皮脂が過剰に分泌される → 毛穴が開いて目立ちやすくなる / 引き締めながら整えることが必要！
3解決策＋成分リスト引き締め成分がカギ / ✓ サリチル酸（BHA） → 毛穴の詰まりを溶かして落とす / ✓ 収れん成分 → 毛穴をキュッと引き締める / ✓ クレイ成分 → 余分な皮脂を吸着してキメを整える / 引き締めながら整えるアプローチ。
4未来＋メリット列挙＋CTA皮脂のリズムが整うだけで、肌が変わる。 / 皮脂のリズムが整うと… / ✓ 鏡を見ても毛穴が気にならなくなる / ✓ キメが整って、つるんとした質感になる / ✓ 皮脂によるメイク崩れが減る / 皮脂に振り回されない、引き締まった肌へ。 / ボタン「あなたに合う化粧水を見る →」
結果カード（4枚×4タイプ）には商材名・価格は一切出しません。Card4のCTA「あなたに合う化粧水を見る→」タップ後、別途共有するPRカルーセルで初めて商材（タイプ内の該当商材を並列）が登場します。

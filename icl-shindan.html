診断設計（Q1〜Q4）

所要時間: 約30秒（4問）／ Q3の回答で結果タイプが分岐

Q1
今、視力矯正をどうしていますか？
最軽入口・自分ごと化
  • A コンタクトレンズを使っている
  • B メガネを使っている
  • C コンタクトとメガネを使い分けている
  • D 何もしていない（裸眼）
Q2
目のことで一番困っていることは？
ペイン把握・自分ごと化
  • A 視力がどんどん悪くなっている気がする
  • B コンタクト・メガネが面倒・不便
  • C 目が疲れやすい・乾く
  • D 左右で見え方が違う・ピントが合いにくい
Q3 — 分岐キー
視力矯正について、今の気持ちに近いのは？
ニーズ・温度感の把握 → 結果タイプ分岐
  • A できれば自力でなんとかしたい → Type A
  • B 眼鏡・コンタクトをやめたいが手術は不安 → Type B
  • C 手術以外の選択肢を探したい → Type C
  • D 手術も含めて根本的に治したい → Type D
Q4
もし目の悩みが解決したら、一番変わってほしいことは？
理想の未来・感情接続
  • A 朝起きたらすぐ見える生活がしたい
  • B スポーツや旅行・温泉を、コンタクトやメガネなしで楽しみたい
  • C ドライアイや目の疲れから解放されたい
  • D 目のことを気にせず過ごしたい

結果カルーセル（4枚 × 4タイプ）

Card1・Card3は全タイプ共通。Card2（不安喚起）・Card4（欲求）がタイプ別。
Type A — 自力探索タイプ
Q3: 「できれば自力でなんとかしたい」
Card2 見出し
「ちゃんと続けてるのに、なぜ変わらないんだろう」
→ 仮性近視と軸性近視の違いを解説。成人の大半は軸性近視でトレーニングでは戻らない。
Card4 見出し
朝起きたらすぐ見える生活、諦めなくていいかも
CTA: 自分の目の選択肢をLINEで確認する（無料）→
Type B — 矯正チェンジタイプ
Q3: 「眼鏡・コンタクトをやめたいが手術は不安」
Card2 見出し
「手術せずにコンタクトをやめる方法」は、実は少ない
→ オルソK・眼内コンタクト等の選択肢と限界を整理。
Card4 見出し
「手術が怖い」気持ち、正直だと思う
CTA: ICLについて、もう少し知ってみる（無料）→
Type C — 手術なし探索タイプ
Q3: 「手術以外の選択肢を探したい」
Card2 見出し
「手術なし」で視力を根本改善できる方法は、今のところない
→ オルソKの限界・コンタクト長期リスク（角膜内皮細胞減少）を提示。
Card4 見出し
ICLは「手術」のイメージと少し違うかもしれない
CTA: 怖くない理由だけ、見てみる（無料）→
Type D — 根本解決タイプ
Q3: 「手術も含めて根本的に治したい」
Card2 見出し
根本的に治す選択肢はある。でも全部が自分に合うわけじゃない
→ レーシック・ICL・PRKの違いと適応条件を整理。
Card4 見出し
目のことを気にせず過ごせる生活、ICLなら近い
CTA: 自分がICLに適応してるか、無料で確認する →

Card3（全タイプ共通・ものさし変換）

見出し: 続かないのは、意志の問題じゃない

近視は目の構造の問題なので、トレーニングや目薬では変えられない。でも医療なら、変えられる。

・角膜を削らない（レーシックとは別物）
・取り出せる（永続的な変更ではない）
・手術時間は約10〜15分

「怖い手術」のイメージは、10年以上前の話かもしれません。

ものさし変換

今のものさし（壊す）新しいものさし（叩き込む）
目が悪いのは仕方ない。眼鏡・コンタクトで一生付き合っていくもの 視力は治せる。角膜を削らず・取り出せる・10分で終わる。眼鏡・コンタクト以外の選択肢がある
ハンマー: 近視は目の構造の問題なので、トレーニングや目薬では変わらない。視力は医療の力で治せる。
